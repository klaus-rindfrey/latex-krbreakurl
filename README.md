# luabreakurl

**Breakable URLs with configurable continuation marks for LuaLaTeX**

`luabreakurl` is a LuaLaTeX package for typesetting long URLs and URIs with automatic line breaking while preserving clickable hyperlinks.

Its distinguishing feature is the ability to display a configurable continuation mark only when a line break actually occurs.

## Documentation

The complete user documentation and documented implementation are available in
`luabreakurl.pdf`, generated from `luabreakurl.dtx`.
